AdventureWorks Försäljningsanalys - SQL & Python
Detta projekt extraherar och analyserar försäljningsdata från AdventureWorks-databasen för att identifiera kopplingar mellan omsättning, vinstmarginaler och lagerstatus. Fokus ligger på att skilja mellan faktiskt realiserat resultat och teoretiskt lagervärde.

För en fördjupad genomgång av projektets resultat och slutsatser, se filen: Analysis_Report

Teknikstack
SQL (Dataextraktion)

Python & Pandas (Analys)

Seaborn & Matplotlib (Visualisering)

SQLAlchemy (Databaskoppling)

Kom igång med projektet
Följ dessa steg för att köra analysen lokalt:

Klona repot genom att använda kommandot: git clone https://github.com/JakobWorkingHard/AdventureWorks.git

Installera nödvändiga bibliotek via pip: pip install -r requirements.txt

Öppna och kör filen connection_docker.ipynb i VS Code eller Jupyter Lab för att generera analysen.
